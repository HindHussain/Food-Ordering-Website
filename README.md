# Food-Ordering-Website
# Description:
This project is a simple and user-friendly website that allows customers to browse a restaurant menu, view food items with prices, and place orders online quickly and easily.
# Problem:
Many customers face delays when ordering food due to long waiting times and the lack of an efficient online ordering system.
# Sloution:
This project is to create a simple to use web application so that customers can view a restaurant menu and easily order food off of that same menu.
# Goal:
The goal of this project is to make food ordering faster, easier, and more organized for both customers and restaurants.

# The website consists of the following pages:
# 1- Home Page:
Displays the restaurant name and a button to view the menu.
# 2- Menu Page:
Page to shows a list of food items with prices and an option to add items to the cart.
# 3- Cart Page:
Displays selected items and the total price.
# 4- Checkout Page:
To allows users to enter their name and phone number to place the order.
# 5- Confirmation Page:
View to Shows a message confirming that the order has been placed successfully.
